# Projeto – Atividade Parcial de Programação Orientada a Objetos (POO)
### Descrição do projeto:
##### Este repositório contém a implementação completa do Trabalho Vivencial da disciplina de Programação Orientada a Objetos. O presente trabalho utiliza a linguagem Java e explora os principais conceitos da linguagem, como o paradigma da orientação a objetos, o encapsulamento, abstração, herança e polimorfismo. Além disso, este projeto também comtempla a modelagem do sistema por meio de um diagrama de classes, seguido pelo desenvolvimento das mesmas. O principal objetivo desta atividade é o desenvolvimento da capacidade dos envolvidos de analisar um problema, modelar sua solução utilizando a Programação Orientada a Objetos.
